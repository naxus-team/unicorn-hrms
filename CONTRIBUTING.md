# Contributing to Unicorn HRMS

Thank you for your interest in Unicorn HRMS. As this is proprietary commercial software, contributions are managed internally by the Naxus development team.

## For Naxus Team Members

### Development Workflow

1. Create feature branch from `develop`
```bash
   git checkout -b feature/feature-name
```

2. Follow coding standards (see docs/development/coding-standards.md)

3. Write tests for new functionality

4. Submit pull request for review

5. Ensure CI/CD passes
