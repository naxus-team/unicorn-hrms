# Unicorn Desktop HRMS - OpenGL Edition

## Quick Start

1. Download dependencies:
   - GLFW from vendor setup
   - GLAD from glad.dav1d.de
   - GLM from GitHub

2. Build:
   ```
   scripts\build.bat
   ```

3. Run:
   ```
   scripts\run.bat
   ```

## Project Structure
- src/core/ - Application core
- src/renderer/ - OpenGL rendering
- src/ui/ - Custom UI system
- vendor/ - Third-party libs
