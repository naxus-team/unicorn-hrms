# Changelog

All notable changes to Unicorn HRMS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-15

### Added
- Initial release of Unicorn HRMS
- Core attendance tracking with GPS geo-fencing
- Leave management system with approval workflows
- Performance evaluation module with KPI tracking
- Employee self-service portal
- Mobile applications for iOS and Android
- Web desktop interface
- Admin dashboard with analytics
- CRM integration capabilities
- RESTful API with comprehensive documentation
- Multi-language support (English and Arabic)
- Role-based access control
- Automated reporting system

### Security
- JWT-based authentication
- Data encryption at rest and in transit
- Comprehensive audit logging
- Role-based permissions system

- **Minor version (0.X.0):** Backward-compatible functionality
- **Patch version (0.0.X):** Backward-compatible bug fixes
