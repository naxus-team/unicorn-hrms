
**Q: Why was my leave request rejected?**
A: Check the rejection reason in the request details or contact your manager.

**Q: Can I see my team members' attendance?**
A: Only if you're a manager. Regular employees see only their own data.

**Q: How often are payslips uploaded?**
A: Payslips are typically available by the 5th of each month.

**Q: I'm having technical issues. Who do I contact?**
A: For system issues, contact support@naxus.com

## Quick Reference

### Common Tasks

| Task | Location |
|------|----------|
| Clock In/Out | Dashboard or Mobile App |
| Request Leave | My Leave > Request |
| View Balance | My Leave > Balance |
| View Payslip | Payroll > My Payslips |
| Update Profile | My Profile > Edit |
| View Attendance | My Attendance |
| Check Goals | My Performance > Goals |

### Keyboard Shortcuts

- `Alt + D` - Dashboard
- `Alt + A` - My Attendance
- `Alt + L` - My Leave
- `Alt + P` - My Profile
- `Ctrl + /` - Search
