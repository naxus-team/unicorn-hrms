# HR Manager Guide

## Overview

As an HR Manager, you have access to employee management, leave administration, performance tracking, and reporting functions.

## Dashboard

Your dashboard shows:
- Pending leave requests
- Employee anniversaries
- Upcoming reviews
- Department statistics
- Recent activities

## Employee Management

### Viewing Employee Records

1. Navigate to **Employees > All Employees**
2. Use filters:
   - Department
   - Status (Active/Inactive)
   - Employment type
3. Click employee name for details

### Adding New Employees

1. Go to **Employees > Add New**
2. Fill Personal Information:
   - Full name
   - Date of birth
   - National ID
   - Contact details
3. Employment Details:
   - Department
   - Position
   - Hire date
   - Manager
   - Salary (if authorized)
4. Click **Save**

### Updating Employee Information

1. Find employee record
2. Click **Edit**
3. Update necessary fields
4. Click **Save Changes**

### Employee Documents

Manage documents:
- Upload contracts
- Store certificates
- Maintain ID copies
- Track document expiry

## Leave Administration

### Processing Leave Requests

1. Go to **Leave > Pending Requests**
2. Review request details:
   - Employee name
   - Leave type
   - Dates
   - Reason
   - Remaining balance
3. Options:
   - **Approve** - Grant leave
   - **Reject** - Deny with reason
   - **Request Info** - Ask for clarification

### Checking Leave Balances

1. Navigate to **Leave > Balances**
2. View all employees' leave status
3. Filter by department
4. Export report if needed

### Managing Leave Policies

Configure policies:
- Annual leave allocation
- Sick leave rules
- Emergency leave
- Unpaid leave limits

### Leave Calendar

View **Leave > Calendar** to see:
- Who is on leave
- Upcoming leaves
- Department coverage
- Conflicts

## Attendance Monitoring

### Daily Attendance

1. Go to **Attendance > Daily View**
2. See real-time status:
   - Present
   - Absent
   - Late
   - On leave
3. Export daily report

### Attendance Reports

Generate reports:
- **Reports > Team Attendance**
- Select date range
- View summary or detailed view
- Export to Excel/PDF

### Leave Reports

Track leave usage:
- **Reports > Team Leave**
- See patterns and trends
- Identify potential issues
- Plan for busy periods

### Performance Overview

Access team performance:
- **Reports > Performance**
- View aggregate scores
- Compare team members
- Track improvement

## Workload Management

### Assigning Tasks

If integrated with task management:
1. Navigate to **Team > Tasks**
2. Click **Assign Task**
3. Select team member
4. Set priority and deadline
5. Track progress

### Monitoring Capacity

Check team workload:
- Current assignments
- Upcoming deadlines
- Available capacity
- Resource allocation

## Communication

### Team Announcements

1. Go to **Team > Announcements**
2. Click **New Announcement**
3. Write message
4. Select recipients (whole team or specific members)
5. Post

### Direct Messages

Communicate with team members:
- Use built-in messaging
- Send individual or group messages
- Share updates and information
- Keep records of communications

## Requesting Resources

### Additional Staff

1. Navigate to **Requests > New Request**
2. Select "Staff Request"
3. Provide:
   - Position needed
   - Justification
   - Timeline
   - Budget implications
4. Submit to HR

### Training Requests

Request training for team:
1. Go to **Team > Training**
2. Identify training needs
3. Submit request with details
4. Track approval status

## Emergency Situations

### Reporting Incidents

If workplace incident occurs:
1. Go to **Reports > Incident**
2. Document immediately
3. Include all relevant details
4. Notify HR and appropriate parties
5. Follow up as needed

### Employee Concerns

If team member raises issue:
1. Listen carefully
2. Document conversation
3. Escalate to HR if needed
4. Follow company procedures
5. Maintain confidentiality

## Tips for Success

1. **Regular check-ins** - Meet with team regularly
2. **Be fair** - Apply policies consistently
3. **Communicate clearly** - Keep team informed
4. **Lead by example** - Model desired behavior
5. **Recognize achievements** - Celebrate successes
6. **Address issues early** - Don't let problems grow
7. **Support development** - Invest in team growth
8. **Balance workload** - Distribute work fairly
