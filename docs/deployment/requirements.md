# System Requirements

## Server Requirements

### Minimum Requirements

**Operating System:**
- Linux: Ubuntu 20.04 LTS or later, CentOS 8 or later
- Windows: Windows Server 2019 or later

**Hardware:**
- CPU: 4 cores (2.5 GHz or higher)
- RAM: 8 GB
- Storage: 100 GB SSD
- Network: 100 Mbps connection

**Software:**
- .NET 8.0 Runtime
- PostgreSQL 14+ or MySQL 8.0+
- Nginx or IIS (for web server)

### Recommended Requirements

**Hardware:**
- CPU: 8 cores (3.0 GHz or higher)
- RAM: 16 GB
- Storage: 250 GB SSD (RAID configuration)
- Network: 1 Gbps connection

## Client Requirements

### Web Application

**Browser Support:**
- Chrome 100+
- Firefox 100+
- Safari 15+
- Edge 100+

**Screen Resolution:**
- Minimum: 1280x720
- Recommended: 1920x1080

### Mobile Applications

**iOS:**
- iOS 13.0 or later
- iPhone 6s or newer
- iPad Air 2 or newer

**Android:**
- Android 8.0 (API level 26) or higher
- 2 GB RAM minimum
- 100 MB free storage

## Network Requirements

- Static IP address (for server)
- SSL certificate for HTTPS
- Firewall configuration for required ports
- Backup internet connection (recommended)

## Database Requirements

**PostgreSQL:**
- Version 14.0 or higher
- UTF-8 encoding
- Sufficient storage for data growth

**MySQL:**
- Version 8.0 or higher
- InnoDB storage engine
- UTF-8mb4 character set

## Security Requirements

- Firewall enabled
- Regular security updates
- Antivirus software (recommended)
- Backup solution in place
- SSL/TLS certificates

## Scalability Considerations

For organizations with:
- **< 100 employees:** Minimum requirements sufficient
- **100-500 employees:** Recommended requirements
- **500+ employees:** Contact Naxus for enterprise sizing
