# Unicorn HRMS

> Enterprise-grade Human Resources Management System by Naxus

![Unicorn HRMS Logo](assets/logos/unicorn-logo.png)

## Overview

Unicorn HRMS is a comprehensive human resources management solution designed for modern enterprises. Developed exclusively by Naxus, it provides integrated tools for attendance tracking, performance management, leave administration, and seamless CRM integration.

## Key Features

- **Smart Attendance Tracking** - GPS-based clock-in/out with geo-fencing
- **Performance Management** - KPI tracking and 360-degree reviews
- **Leave Management** - Automated workflows and balance tracking
- **Mobile Applications** - Native iOS and Android apps
- **CRM Integration** - Seamless customer interaction tracking
- **Advanced Analytics** - Real-time reporting and dashboards
- **Customizable Workflows** - Tailored to your business processes

## Technology Stack

- **Backend:** .NET C# with Entity Framework
- **Database:** PostgreSQL/MySQL
- **Frontend:** Modern web technologies
- **Mobile:** Native iOS and Android
- **API:** RESTful architecture with JWT authentication

## System Requirements

### Server
- OS: Linux (Ubuntu 20.04+) or Windows Server 2019+
- RAM: 8GB minimum (16GB recommended)
- Storage: 100GB SSD
- .NET 8.0 Runtime

### Client
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Mobile: iOS 13+ or Android 8+

## Installation

Detailed installation instructions are available in the [Installation Guide](docs/deployment/installation.md).

Quick start:
```bash
# Clone repository (requires access)
git clone https://github.com/naxus-team/unicorn-hrms.git

# Follow setup instructions
cd unicorn-hrms
./scripts/setup/install.sh
```
